KIT - Keep It Tidy

KIT is a task manager tool developed in Java and JavaFX for the module CS2103 - Software Engineering.

KIT is a desktop app that helps users manage their tasks and it is the one and only taskmanager you will ever need. With a simplistic design of UI and powerful functionalies embedded, KIT is capable of executing your instructions swiftly. On top of that, KIT is keyboard friendly, thereby saving you the trouble of ever moving the mouse and clicking.
Features in charge of: Storage, UI/UX - theme changes, tag colors, placements of containers.

SPA - Static Program Analyzer

SPA is a static program analyzer which acts as the first step of a compiler for the SIMPLE language developed for CS3201/02 Software Engineering Project I & II.

A Static Program Analyzer (SPA for short) is an interactive tool that automatically answers queries about programs. Of course, an SPA cannot answer all possible program queries. But the class of program queries that can be answered automatically is wide enough to make an SPA a userful tool.
Features in charge of: Storage, Knowledge-based system, Design Extractor, API formulation.

NUS Battleground

NUS Battleground is a location-based mobile game developed for the Orbital Summer Software Engineering project.

NUS Battleground is a location-based mobile game developed on AngularJS, HTML and CSS using the Cordova framework. The objective of the game is to collect as many items which will aid your battle against a boss which will spawn at various locations around the NUS Kent Ridge campus.
Features in charge of: UI/UX, database, program logic.

A simple website developed for The T Project in the module NM3217 Design for Strategic Communications.

The T Project is Singapore's only shelter for homeless transgenders. The website is designed with the objective of raising funds to help support the shelter, as well as raise awareness of homeless transgenders in Singapore.
